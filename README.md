# forex_analisys ИИ, регрессия

В целом здесь сырой проект по анализу рынка форекс, есть дашборд, вывод в консоль прогнозов, сохрание результатов.
Все основные файлы в папке '''Python'''

Здесь представлен проект по фин теху, работа с рынком форекс. 

Обучал модель LSTM - регрессия. 

За основу взята валютная пара EUR/USD, таймфреймы: 15м, 30м, 1Н, 4Н, 1Д. 

Работал от сбора данных, нанесения индиктаторов, создания целевой метки, обучение модели и вывод графиков

Наблюдения: Странно что на первых моделя сходимость loss и val_loss была широкой, но результат модель выдалава 50/50 что даже привело к заработку на демо счёте. 
Но стоило добавить индикторы, уменьшил слои(это на тф 1Н) как модель показала лучшую сходимость, но резульаты цены космические.

Обучение проводил первых моделей локально, последующие проводил в колабе.
